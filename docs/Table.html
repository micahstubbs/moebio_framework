Class: Table

Table

A sub-class of List, Table provides a 2D array-like structure.

Each column is stored as its own List, making it a List of Lists. Cells in the table can be accessed using table[column][row].

Additional functions that work on Table can be found in:

  • Operators: TableOperators
  • Conversions: TableConversions
  • Generators: TableGenerators
  • Encodings: TableEncodings
