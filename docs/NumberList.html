Class: NumberList

NumberList

List structure for Numbers. Provides basic data type for storing and working with numbers in a List.

Additional functions that work on NumberList can be found in:

  • Operators: NumberListOperators
  • Conversions: NumberListConversions
  • Generators: NumberListGenerators

add(object) → {NumberList}

Adds a value or values in a NumberList to the current list.

If input is a Number, each value of the returned NumberList will be the sum of the original value and this input value.

If the input is a NumberList, each value of the returned NumberList will be the sum of the original value and the value at the same index in the input list.

getSortIndexes(descending) → {NumberList}

Returns a new NumberList containing the indicies of the values of the original NumberList in sorted order.

Parameters

Name Type Description
descending Boolean

If true, values are sorted in descending order.

Returns NumberList

NumberList containing the indices of the original NumberList such that accessing the values of the original list at those indices would produce a sorted list.

Example

var nl = NumberList.fromArray([1,3,2]);
var indices = nl.getSortIndexes();
indices[0]; // produces 1 as nl[1] == 3.
