Namespace: NetworkGenerators

NetworkGenerators

NetworkGenerators provides a set of tools to generate Network instances from a variety of sources.

Methods

(static) createNetworkFromListAndFunction(list, weightFunction, names) → {Network}

Creates a network using a list and measuring the relation weight with a given method a Relation is created between two nodes if and only if the returned weight is > 0

Parameters

Name Type Description
list List

List of objects that define the nodes
weightFunction function

method used to eval each pair of nodes
names StringList

optional, names of Nodes

Returns Network

a network with number of nodes equal to the length of the List

(static) createNetworkFromOccurrencesTable()

(static) createNetworkFromTextAndWords(text, nounPhrases, splitCharacters) → {Network}

Builds a network from a text, using previously detected words or noun phrases, and with relations built from co-occurrences in sentences relations contain as description the part of the sentence that ends with the second node name (thus being compatible with NoteWork)

Parameters

Name Type Description
text String
nounPhrases StringList

words, n-grams or noun phrases
splitCharacters String

split blocks by characters defined as string regexp expression (defualt:".|\n"), blocks determine relations

Returns Network

tags:

(static) createRandomNetwork(nNodes, pRelation, mode, randomRelationsWeights, seed) → {Network}

Build a random network based on the provided options

Parameters

Name Type Description
nNodes Number

number of nodes
pRelation Number

probability of a relation being created between 2 nodes
mode Number

0:simple random 1:clusterized
randomRelationsWeights Boolean

adds a random weigth to relations
seed Number

random seed for stable random generation

Example

// generate a sparsely connected network with 2000 Nodes
network = NetworkGenerators.createRandomNetwork(2000, 0.0006, 1);
