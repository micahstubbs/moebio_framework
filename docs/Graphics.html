Class: Graphics

Graphics

A graphics object provides a surface for drawing and interaction with drawn primitives.

Constructor

new Graphics(options, noLoop, noStart)

Creates a new graphcis object and initializes basic configuration and optionally starts the draw cycle.

Parameters

Name Type Description
options Object

valid properties described below

Properties
Name Type Description
container String | DOMNode | undefined | false

a string selector or reference to a node where the canvas for this graphics object will be placed. If this is a falsy value, then the graphics object will not be attached to the DOM and can be used as an offscreen bugger.
dimensions Object

object with width and height properties to specify the dimensions of the canvas. If this is not passed in, the canvas will grow to always fit the size of its container.
init function

custom user function to initialize drawing related things. Called once after base initialization.
cycle function

custom user function to render each frame. This is called once per frame of the draw loop unless cycleInterval is set to 0.
onResize function

custom user function to run on resize of the canvas *
cycleInterval Number

how often in muilliseconds to run the cycle function if set to zero the cycle function will only run once.
noLoop Boolean

if this is true only run the cycle function once,
noStart Boolean

if this is true the cycle function will not be automatically started.

Members

WHEEL_CHANGE

Mozilla case.

Methods

_cycleFor(time)

Run the cycle function for a certain amount of time and then stop it.

Can be called multiple times to delay the stop time to being time milliseconds after the last call to this function.

Parameters

Name Type Description
time Number

time in milliseconds to run the cycle function before stopping ot.

setStroke(style, lineWidth)

Set stroke to draw with in canvas

Parameters

Name Type Description
style String | Number

If string, then hex value or web color. If Number, then a set of RGB or RGBA integers
lineWidth Number

Optional width of line to use. Only valid if style parameter is a string.

Example

setStroke('steelblue'); // sets stroke to blue.
setStroke(0,0,0,0.4); // sets stroke to black with partial opacity.
setStroke('black', 0.2); // provides lineWidth to stroke
